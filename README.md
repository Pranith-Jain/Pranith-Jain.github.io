# Pranith Jain - Portfolio

Static portfolio website built with HTML, JavaScript, and Tailwind CSS.

## Features
-   Responsive design
-   Dark mode support
-   Clean and modern UI
-   Static and lightweight

## Hosting on Cloudflare Pages
1.  Upload `index.html` and `script.js` to a new Cloudflare Pages project.
2.  No build step required.
